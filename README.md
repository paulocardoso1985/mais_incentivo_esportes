# Mais Incentivo Esportes

Plataforma de fidelidade e incentivo voltada para o mercado de esportes, permitindo que lojistas e vendedores acumulem **Mais Coins** através de compras no Grupo Votorantim e os troquem por experiências inesquecíveis em estádios de futebol.

## 🚀 Funcionalidades Principais

- **Dashboard Inteligente**: Acompanhamento de metas e insights de venda para Gerentes Regionais.
- **Regionalização**: Experiências e prêmios filtrados por localização para maximizar o engajamento local.
- **Logística Integrada**: Planejamento de viagens com detalhes de voos, hotéis e conexão diretamente no resgate.
- **Gamificação**: Sistema de "Próximo Resgate" para motivar o vendedor de balcão.

## 🛠️ Tecnologia

- **Frontend**: Next.js, Tailwind CSS.
- **Backend**: NestJS, Prisma ORM.
- **Banco de Dados**: PostgreSQL (Docker-ready).
- **Monorepo**: TurboRepo.

## 📦 Como rodar (Desenvolvimento)

1. Clone o repositório.
2. Suba o banco de dados: `docker-compose up -d`.
3. Instale as dependências: `npm install`.
4. Rode o projeto: `npm run dev`.
